# OBSIDIAN Acquisition & Strategic Licensing Platform

An ultra-premium, high-fidelity acquisition landing page for **OBSIDIAN** (Adaptive Identity & Ambient Intelligence platform). This platform represents a next-generation security paradigm that fuses multimodal biometrics (face, voice, and behavioral analytics) for zero-password continuous verification.

## 🌟 Core Architecture & Strategy

- **Adaptive Intelligence**: OBSIDIAN goes beyond traditional identification to understand human states (e.g., calm, stressed, anxious, aggressive) and dynamically adapt trust weighting and risk scores.
- **Luxury Security Persona**: Positioned specifically for high-end luxury venues, hospitality sectors, retail, and security HUD command centers.
- **Preserved Strategic Briefing**: Integrates "The Founder's Briefing" podcast player providing deep-tech analysis of the underlying cloud transition and platform deployment architecture.

## 📊 Live Interactive Sections

1. **Competitor Comparison Matrix**: Dynamic, glassmorphic layout highlighting OBSIDIAN's technical superiority over Okta, Ping Identity, IDEMIA, iProov, FaceTec, CLEAR, and Megvii.
2. **Deep-Tech Valuation Trajectory**: Interactive tracking of target valuations ranging from early-stage prototypes ($500K-$2M) up to global category-defining platforms ($100M+).
3. **Capabilities Matrix**: Capabilities comparison detailing traditional MFA vs enterprise IAM vs OBSIDIAN.
4. **Interactive Lightbox Modal**: Click to overlay full-resolution architectural diagrams (`infographic-1.png` and `infographic-2.png`) directly in-browser.

## 🚀 Deployment & Local Testing

Simply open `index.html` in any modern web browser to interact with the responsive layouts, transition animations, and custom audio player components.
